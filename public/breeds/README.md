# Race-billeder

Læg et billede her for hver race, navngivet efter racens **slug** + `.jpg`.
Fx `labrador.jpg`, `border-collie.jpg`, `maine-coon.jpg`.

- Format: **JPG**, ca. **800×600 px**, optimeret (helst < 150 kB).
- Ingen kodeændring nødvendig — filen vises automatisk på race-kort,
  beregner og detaljeside.
- Mangler en fil, vises en pæn emoji-placeholder (🐕/🐈) i stedet — intet
  fremstår "brudt".
